## 功能

- 支持关键词搜索
- 支持过滤图标库
- 左键点击：复制图标名称
- 右键点击：打开菜单
  - 复制 import 语句
  - 复制组件名称
  - 复制 SVG 代码
- 支持系统暗黑模式
- 快捷键 （开发中）
  - `Ctrl + F`  `Ctrl + K`  `/`：搜索图标
  - `Tab`: 下个图标
  - `Ctrl` + `Tab`: 上个图标
  - `Enter`: 复制图标名称
